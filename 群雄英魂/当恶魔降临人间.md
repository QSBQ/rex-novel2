# 锲子

```
   一处破旧的废墟里，被战争严重摧残的一栋破楼中，一个勉强能住人的房屋内。
   一名少年穿着破旧的衣物，颓废的躺在破碎的沙发上。
   他手里拿着一瓶碑酒，咕噜咕噜的往嘴里倾倒。
   “嗝~呵——”
   他放下酒瓶，一副醉生梦死的表情，痴迷于幽寂暗黑环境的软泥身体。
   咚咚！咚咚！
   门外传来了敲门声。
   “谁——啊！”
   他不耐的，用尽全身力气似的拖着自己懒惰的喉咙完成这一声艰难的呼叫。
   “我，穆斯将军”
   一道严肃中年人的浑厚声音透过屋门传了进来。
   少年听见门外的回应，颓废的意识稍稍清醒了些，稍稍有力了些，身体像是触电般动弹了下。
   但很快又陷入死寂。
   咚咚！咚咚！
   敲门声再次传来，一如既往的有力。
   少年不耐烦的，极为勉强的把自己从沙发上拖起，背靠着沙发仰头望着天花板。
   “你·妈·逼”
   他吐出这三个字，为不速之客的来访而极不高兴。
   艰辛的挪动自己的脚步，刚挪一步就差点摔倒，身体一抽一抽的，姿势诡异。
   终于开了门。
   一个面容凌厉，身材高大，左眼到嘴边有一道深切伤疤的绿色军装中年人站在少年的面前。
   他的身边有两个跟随者，一个瘦高个男性，一个小麦肤色的光头，也都穿着军装。
   “干什么！”
   少年不耐烦的怒吼。
   穆斯将军没有因为少年的怒火而动怒，只是严正的开口道：
   “苏玄，我希望能与你再一次达成合作”
   “什么狗屁合作，找别人去！”
   苏玄摆摆手，转身回到沙发坐下。
   “小子！我警告你……”
   穆斯将军还未说话，他身边那个小麦肤色的光头早已忍不住踏前一步，大声喝斥，但话未说完，却戛然而止。
   “警告我？”
   苏玄转头，眼眸精光爆射，掌心延伸而出的一根腥红触手紧紧缠住光头的喉咙，一个颓废的少年瞬间变得无比恐怖。
   光头大惊失色，全身黄金硬化，双手泛起耀眼的金光，抓住触手想要撕开，但触手却坚韧非常，根本不为之所动，反而越勒越紧，黄金颈项都快要被撕裂粉碎！
   他大骇，感受到了致命的死亡威胁。
   要知道，他可是飞廉帝国的金牌锐士，绰号“巨力金刚”的李庭望，如今却被一根触手轻易的致于将死的境地。
   “他是我新收的部下，不懂规矩，希望你能手下留情，饶过他的性命”
   穆斯将军诚挚道。
   “哼！”
   触手一甩，李庭望登时倒飞出去，一连撞穿数栋大楼，砸入废墟，发出巨大声响。
   一片烟尘滚滚，瘦高个男性额间冷汗直流，他叫张界石，同样是飞廉帝国的金牌锐士，绰号“炸弹手”，通过触摸，每根手指都可以将触摸过的物品变为一种炸弹，威力最小的一种都能炸毁数座房屋，但此时此刻，他却不敢在这个少年面前轻举妄动。
   “什么时候你也会收这么废物的部下了？”
   苏玄的左手食指变作一根触手，将桌上的一罐碑酒摄来，拧开，咕噜咕噜的饮用。
   “废物也有废物的利用价值，与‘税皇’一战，我们损失惨重，只好以废物滥竽充数”
   穆斯将军的语气很冷漠，将那个激怒苏玄的部下直呼为废物。
   苏玄翘着二郎腿，一副玩世不恭的模样。
   “皇甫铳啊，他的‘纳税人队伍’壮大到多少人了？我真是好久没跟他动过手了”
   穆斯将军依旧站立着，神情严肃，回应道：
   “他已经有了三百万‘纳税人’，他的实力已经达到了一个相当恐怖的地步，我们飞廉帝国的异能者和修武者全部一起上也只勉强的把他逼退。”
   苏玄的手指在桌上轻轻敲打。
   税皇皇甫铳，通过向异能者或修武者“收税”，可以将异能者或修武者变为自己的“纳税人”，能够抽取异能者或修武者的部分力量甚至是部分异能为自己所用，但相应的，如果“纳税人”死亡、受伤，甚至只是心情变差，他的力量都会受到影响。
   “你们是想让我出手打败皇甫铳？按照目前形势来看，皇甫铳的纳税人再多个七八十万的话，你们只有亡国的份了”
   出手打败皇甫铳？！
   张界石眼角一缩，心脏怦怦乱跳，眼前这个其貌不扬的颓废少年，竟然是能够与税皇那种强大的怪物平起平坐的存在吗？
   他看了一眼穆斯将军，这名将军神色不变，竟然默认了这种说法，这令他心惊不已。
   也许，坐在自己面前的这个少年。
   不！这位大人。
   将会是自己一生至今为止遇到过的最强怪物！
   “我的国王已经与税皇达成协议，建立条约，决定维持和平的局面长达三年，此事暂时了结，我来找你，是为了另一件事”
   穆斯将军的语气诚恳了些许，他迫切的希望苏玄能够答应他的请求。
   苏玄不以为意，只吐出一个字：“滚！”
   穆斯将军似乎早已预料到苏玄会是这个反应，直接提出了自己手中的筹码：
   “我们知道你的姐姐——苏诺曦，她最后失踪的地方”
   苏玄瞳孔紧缩，一转头，周身气势疯狂推至顶峰，凶戾的如死神一般恐怖，死死的盯着穆斯将军：
   “你，真的知道？”
   空气都仿若为之凝固，张界石感觉自己正面对一头恐怖的史前怪物，又像是溺水了一样，瞬间憋红了脸，简直快要窒息！
   穆斯将军面色绷紧，正面承受着一股迫切的压力，但他终究是见过大风大浪的，开口说道：“绝无半句虚言，情报来源于我们帝国最高情报组织——暗台，错不了”
   “这是报酬？”
   苏玄的气势松缓下来，穆斯将军心下松了一口气，张界石大口大口的喘着粗气。
   穆斯将军先让张界石退下，然后慎重说道：“不是报酬，这将是委托任务的定金，我们希望你能够护送陛下的女儿，帝国的公主，前往黑冰山脉”
   “那真正的报酬是什么？”
   苏玄的眼中涌现出强烈的渴望。
   苏诺曦，是他从小就跟着流浪的青梅竹马，缔结约定的义姐，感情极深，自从苏诺曦在两年前失踪后，他也利用过高科技设备对他的姐姐进行过追踪定位，也找过精通此道的人士帮忙，但无一例外均以失败告终。
   像是人间蒸发了一般，有关于他姐姐的下落的信息像是被一只神秘的大手给抹除。
   难以寻踪。
   “暗台组织以牺牲掉一支十四人小队为代价，找到了苏诺曦的遗留物，我们没有打开，但想必其中有苏诺曦的线索”
   “我姐姐的遗留物，不是理应交给我这个亲友吗？你怎么把它当报酬了？”
   苏玄的眼神微眯，危险的光芒迸发。
   纵是久经沙场的穆斯将军也在这危险的目光下紧张了几分，连忙解释道：
   “苏玄，我们没有将其霸占的意思，但是，首先，你姐姐的遗留物处在一个危险的地方，我们耗费了十四个高级情报人员的生命才将它取得。”
   “其次，如果你不愿意与我达成合作，不愿意接受委托，我们也绝不会霸占你姐姐的遗留物，而是会将它归回原位，就当我们从来没有遇见过它”
   “你看，可以吗？”
   穆斯将军最后的语气，甚至都带上了几分小心翼翼。
   苏玄的气势平缓下来，点了点头，道：“这个委托我接了，说出我姐姐最后失踪的地方”
   能够让穆斯将军如此大费周折，开出一个他无法拒绝的报酬，特别是在面临皇甫铳威胁的前提条件下，他明明可以委托他去打败皇甫铳，但却没有这么做，反而是委托一个护送公主的任务，那么这个任务，绝对不简单，背后涉及的利益定然在“打败皇甫铳”这件事之上，路途也必然充斥着难以预料的危机。
   这是一个极特殊的任务。
   但他不在乎。
   只要能找到姐姐。
   再说，他这个人，艺低人胆大，艺中人胆大，艺高人胆大。
   现在都第三个档次了，完全在自己可以应付的范围，哪有害怕的道理？
   穆斯将军看苏玄答应这个委托，心中一颗大石头落下，说道：
   “黑冰山脉”
   “嗯？”
   苏玄眉头微蹙，一股戾意涌上心头。
   “你应该知道欺骗我的后果是什么！”
   穆斯将军一颗心又提了起来，急忙解释道：
   “这绝非谎言，在四年前，暗台组织的大部分成员跟飞廉皇室研究院的十多位老教授以及一百多位资深研究员，组成考察队前往黑冰山脉进行考察，就是为了公主的黑冰山脉之行。”
   说完，他又特别补充道：“这是帝国的最高机密”
   籍此展现自己的诚意。
   “黑冰山脉，么”
   苏玄若有所思。
   黑暗天，苦寒地，那刺骨的严寒，一般的机器人进去都能被冻死，对妄想仅凭肉体凡胎而走进去的凡人而言，更是绝对的生命禁区。
   更别提除刺骨严寒外，还充斥着各种各样的恶劣异象，陨石般巨大的冰雹，时不时的严重失衡的重力，灭世一般的雪龙卷，还有深藏在苦寒深处的恐怖恶魔……
   这一切的一切，都让黑冰山脉成为了人类的终结地，修为高超的修武者和觉醒强大异能的异能者都极少踏足。
   他不明白苏诺曦为什么要去那里。
   但她既然去了，还消失在了那种危险的地方，他就一定要去找她。
```
