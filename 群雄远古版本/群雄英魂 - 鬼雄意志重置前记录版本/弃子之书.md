# 第五回  冥王金字塔

```
   “如果有吃有喝这种延续生命的最低标准可以被当作好的人生来对待，那这种人生应当终结。
   ——一个天生的残疾人”
   ——《终亡者日记》
   “
   瑯琊，你现在在哪？
   还在山津国吗？
   森罗惧森东南部神秘金字塔现世，异动，根据以往经验，里面的陪葬品价值高昂
   统领世界的各大势力本欲派出强者瓜分其中宝物，但经过探知，这座金字塔的主人是埃及九柱神之一的冥王欧西里斯，其重要性要远远胜于普通金字塔
   因为不知名的原因，冥王金字塔暴动，处于警戒状态，如果派遣魂魄强度极高的强大修士们进入其中，将会触动感应阵法，极有可能引发金字塔自爆
   所以这些举世闻名的强者们经过协商后都一致决定派遣各大势力的天骄去抢夺，谁抢到即是谁的，各大势力的年轻一辈自成一派，成了各大势力战争的缩影，暗中较量的明面战场，某种意义上的代理人战争
   在那一带，祖墨国是当中唯一的大国，隔着森罗惧森的南端、尾佐国，就是你三个月前跟我说的山津国，距离只有几万里，相当邻近，乘坐最好的行者类偃机，哪怕需要转乘数次，一天不到就可以到达
   我将通过白蜮学宫的传送门前往祖墨国的首都鹰皇都，决定夺下金字塔异宝的各大势力的年轻一辈（有许多大势力因为某些不可知的原因，或因其危险性，没有选择拿自己一方的天骄作赌注，或者年轻一辈较为衰弱，拒绝参加这场名为夺宝，实为盗墓，危险性极高的代理人战争，但参加的大势力的天骄数量已经非常惊人），都将汇聚于鹰皇都
   冥王金字塔里的异宝绝对非同寻常，但因为已经被各方大势力圈定的缘故，闲散势力或闲散修士想要进入是不被允许的，你如果感兴趣（里面的异宝绝对值得一争），把你现在所处的地域告诉我，我可以去找你，以我的名义，带你一起去还是可以的，我在白蜮学宫的这次争夺异宝的出行中还是有一定地位的
   无须为此客气，所谓肥水不流外人田，你的助拳对我而言非常重要
   我将于明天经由传送门抵达，这是各大势力协商过后共同认定的日期，亦是各大势力暗中较量的结果，似乎已经发展成为各大势力天骄展露自己和消灭其他大势力的天骄的一个机会了（虽然主要目的和明面目的都直指异宝）
   不管怎样，我希望你能来
   崔道盛
   群雄纪年33年7月14日，金曜日
   ”
   齐瑯琊看完书信，顿时一惊，冥王金字塔？埃及九柱神？
   他一直以为那就是一座普通的金字塔，叫不出名字的埃及历史上某位法老的陵墓！
   齐瑯琊心想，如果说金字塔主人就是冥王本尊，那金黄匣子里的是什么？放在冥王欧西里斯胸前的会是他生前的武器吗？
   远古冥王的武器？！
   思绪万千，心情不可遏制的激动起来，崔道盛是他从前就认识的好朋友，现在奇衡世界的正南之地——久延长洲——极其遥远的距离以外的白蜮学宫中修行，是个实力和背景都很强大的人，如果是崔道盛，那就一定能找到人破解金黄匣子！
   齐瑯琊立即抽出书信卷轴右端的羽毛笔，写下一封回信。
   “
   道盛：
   我现在就在祖墨国的首都鹰皇都，鹤曲楼，柒拾玖房，你一定要过来找我，我有重要的事情要跟你说！
   齐瑯琊
   群雄纪年33年7月14日，金曜日
   ”
   两个奇迹坐标之间，一个月最多只能互通一次书信，所以每封书信谈及重要事情时都要尽可能的详细，但因为明天就可以相遇，所以完全没有问题。
   在更晚些时候，又有一封书信发给了齐瑯琊。
   “
   瑯琊：
   听闻一金字塔现世，牵动各方，各大势力经过协商，不少都决定派出年轻一辈去争夺
   承师父所托，我与师姐二人将是仙虹洞此次出行争夺金字塔异宝的领袖
   明日一早，就率领一众弟子进入传送门，穿梭虚空隧道，降临祖墨国鹰皇都
   你游历天下，三月前曾听你到了山津，不知今时又在何方
   你若对异宝有兴趣，告知我你现今所在，我去寻你
   洛岚歌
   群雄纪年33年7月14日，金曜日
   ”
   齐瑯琊哈哈大笑，立即写了一封回信。
   “
   岚歌：
   现在我就在祖墨国的鹰皇都，鹤曲楼，柒拾玖房，你一定要过来找我，我有重要的事情要跟你说！
   齐瑯琊
   群雄纪年33年7月14日，金曜日
   ”
   接着，齐瑯琊将卷轴收进纳空戒中，端坐在床上，闭眼修炼造根法。
   …………
   7月15日，土曜日，破晓。
   久延长洲，白凤青鸾国，白蜮学宫。
   白玉台上，一名身着黄金铠甲的威严男子庄重宣布道：
   “一切准备就绪，出发！”
   六名担任此次行动的队长的少年少女脚踩白玉阶梯，率先通过前置传送守卫的探知，踏入传送门，他们身后的弟子们也陆续走入，步伐整齐如一。
   ……
   武部隐洲，秦山。
   在山主和各大长老的注视下，一众天才天骄涌入传送门。
   ……
   昼魔境洲，仙虹洞。
   ……
   幽笙界洲，佐佐木氏族。
   …………
   经过一段虚空隧道的航行之后，各队人马均抵达鹰皇都，从一堵巨大的传送门的虚空中走出，祖墨国君主及一干大臣均在此守候，等待着诸位天骄的降临，语气之谦卑，再无往日之神气。
   祖墨国虽然是武部隐洲的一个大国，但一来森罗惧森并非祖墨国领土，那里由龙族统治，一头凶残的霸王龙坐镇，二来就算是在祖墨国领土，单凭祖墨国的实力也占据不住这个宝藏。
   君主无能，依靠麾下一位强大的忠臣安负漠及以安负漠为首的安氏家族的庇佑，得以成为祖墨国的君主而免遭篡位，亦能在祖墨国目前正处于的战争时期中以此为依赖，方才没有让延续了数千年的大国断送在自己的手中。
   如今大势力的天骄们齐聚于此，是一个绝佳的机会，一个能令祖墨国翻身的机会，故态度极为诚恳。
   祖墨国的丞相安负漠设了一场极致豪华的大宴款待各位天骄，邀请诸人进皇宫用膳。
   只有其中一名绝美少女叫来一名女侍者，让其带路前往鹤曲楼，从赶往皇宫的队伍中离开了。
   在她不久后，又有一名少年离开了。
```

# 第六回

```
   “瑯琊，好久不见”
   白衣如雪，背后背着一柄长剑，剑鞘剑柄通体呈银白色，色泽鲜亮，隐隐有波光流动。
   一头银发散落腰间，腰佩一张天鹅形玉牌，上刻仙虹，眉心上一个鹰形刺青，纤纤睫毛下的双目如山洞内的深潭，整个人如同雪山上的厚厚积雪，寒冷而幽寂。
   绝美的少女，她冲着他笑了笑。
   齐瑯琊笑道：“嗯”
   少女坐在了齐瑯琊的床上。
   一个金发青衣，剑眉星目，长相俊美的少年进来了，笑问道：“没有打扰到你们吧？”
   齐瑯琊说道：“打扰到了，赶紧出去！”
   少女说道：“快出去！”
   “呃……”
   齐瑯琊走过去关上门，推上门栓，走到圆桌，解开腰间的金黄匣子，松开青灰色的布，平铺在圆桌上。
   “岚歌，道盛，来看看这个”齐瑯琊的手指敲打在浮雕上，发出“哒哒”的声音。
   洛岚歌走到他身边，目光清冷的看向那金黄匣子。
   “这是何物？”崔道盛问道。
   齐瑯琊得意的笑道：“你这番领队前来是为了何物，它就是何物”
   “冥王异宝？”崔道盛惊讶道。
   洛岚歌转过头，看向齐瑯琊。
   齐瑯琊轻笑道：“你们知道冥王金字塔为什么现世吗？”
   崔道盛惊异道：“我本以为是因为‘水滴石穿’，但没想到另有内情，而且你竟然还知道？”
   “水滴石穿”，是凭空出现的某些东西，或人或物，最常见的原因。
   譬如说有修士封印某物或某人，被封印物自然散发的力量或被封印人不断攻击封印，导致封印在日积月累中被溶解，直到解开，其中的物或人现世。
   这种情况，就被称为“水滴石穿”。
   像金字塔这种陵墓，假若是用某种阵法或利用某种特殊地形将其掩盖或潜藏，那么阵法或地形存在难以察觉的缺陷的话，在时间的河床中就会被水流冲刷而导致缺陷扩大，直到最后的破裂导致其现世。
   “我六月时为了收集毒药的炼制材料途经尾佐国去到森罗惧森……”
   齐瑯琊娓娓道来，将遇到黑袍人、跟踪黑袍人还有盗走法老钥匙的事情全部相告。
   “这就是冥王至宝？”崔道盛眼神炽热的盯着金黄匣子，手指在其上摩挲着，“瑯琊啊瑯琊，你可真是给了我一个大惊喜，我刚领队过来，你就告诉我金字塔现世你占了一半的原因，最大的宝贝竟然就掌握在你的手中！”
   洛岚歌冷静的说道：“既然是钥匙，那它是欧西里斯的武器的可能性微乎其微，更大的可能是它是一个指引的信物” 
   齐瑯琊说道：“现在首先要找到一名偃师来破解金黄匣子”
   两刻钟后。
   “他叫涂解忆，是我带领的小队中偃术最厉害的。”崔道盛介绍道。
```
