# 第一回  僵尸

```
   “人生至难至乐之事：死而无憾”
   ——《黑白孤途》
   黑暗的巨大森林中，一道身影在木叶之间飞奔，怀中抱着一个匣子，他一脚踩在一根巨大树枝上，起身一跃，横跨数十丈，抵达另一头，速度极快。
   清冷幽郁的月光倾斜，映射出的人影轮廓诡异阴邪。
   “咻！”“咻！”“咻！”“咻！”“咻！”
   五道蚕丝射出，银龙飞舞，刺破黑暗，诡异人影在半空一回头，狰狞可怕的面目出现在月光下。
   一对腥红嗜血的竖瞳，阴黑暗寂的眼白，嘴巴撕裂，像是自己亲手用剪刀剪开，近乎裂到尖长的双耳，尖锐獠牙极为可怖，没有鼻子，指爪瘦长锋利，皮肉干瘪，毛发尖细，肤色惨白，全身冰冷、僵硬。
   僵尸！这是一具僵尸！
   “嗖！”
   他向身后甩出一柄锃亮的飞刀，斩破五道蜿蜒击来的蚕丝，“嗤！”的一声扎入树中，脚步不停，向前飞奔。
   六道人影先后从黑暗中窜出，他们身穿宽大黑袍，空洞袍口甩出无尽锁链，相互连结，各自站在六个方位，形成一个巨大的铁锁链网将僵尸封锁在内。
   齐瑯琊站定在一根巨大树枝上，腥红的竖瞳滴溜溜的转动，尽显诡异，瞧见被封锁的球形空间，瞧见维持封锁的六道人影，其宽大黑袍之内，实则是六具受人操纵驭使的无神尸体。
   转身，紧盯着后方的黑暗。
   “天赋神通：傀儡”
   他的嗓声尖锐而沙哑，难听得磨人。
   “交出匣子，我可以放你走”
   一个将自己的身形藏在一顶神秘黑袍之中的人，落在离齐瑯琊最近的一根巨大树枝上，冷声开口道。
   齐瑯琊冷哼道：“这东西我不会放弃的！”
   九天前，他跟踪了眼前这个黑袍人，黑袍人通过某种奇异的秘法，令藏在神秘之地的金字塔现形，开启金字塔，一路破开机关结界，揭开尘封已久的棺椁，将金字塔主人胸前放着的法老钥匙取下，封印进他怀中抱着的这个匣子之中。
   乘着黑袍人全神贯注的将金字塔主人——一具木乃伊，炼制为傀儡的最佳时机，齐瑯琊窃取了匣子。
   狐计不发一语，左手一扬，一具傀儡从他戴在食指上的纳空戒中飞出，铁锁链网同步打开一个窄小的洞口，傀儡窜入，望齐瑯琊手中的匣子抢去。
   血色潮汐从齐瑯琊的身上升起，他的气势瞬间攀升至顶峰，一股恐怖的凶戾气息瞬间爆发，一手探出，五指扣住傀儡头颅，五口铁指插入傀儡脑颅，将其掷在铁锁链网。
   “你当困得住我？”齐瑯琊低吼，向前一跃，撕开冷硬的锁链，向着狐计攻去。
   “锵！”
   一具剑客傀儡飞出，斩出一剑，与齐瑯琊锋利的指刃相触，顿在半空，如针尖对锋芒，互不相让。
   齐瑯琊一脚踢在剑客傀儡的膝盖，借力后空翻，跃在更高处，借着枝叶的遮挡，迅速遁走。
   狐计紧急追上，已不见身影在何方。
   …………
   竖日黎明，昏暗的夜幕现出清冷的蓝天，云作鱼鳞密布，留有细致的间隙。
   齐瑯琊走出名为“森罗惧森”的巨大森林，恐怖阴寒的僵尸体魄收起，恢复人类肉身，本来的面目。
   上半身暗红衣衫，下半身黑色下衣，手脚皆缠绷带，脊背挺直，略显削瘦的身体里仿佛蕴藏着无穷的力量，凶眉狼目，眉宇间自有一股不服输的气质。
   这是一个如狼一般的少年。
   他走进一座小城池，住进一间酒楼。
   他将匣子平放在桌上，仔细端详，手指在匣子上轻轻摩挲。
   金黄银杏色泽，制成的树木极为古老，散发沉重岁月气息，长方体六面无缝，皆有高贵神祇浮雕绘刻，像是镇守邪物。
   金黄匣子无法被纳空戒收纳，其内封存的法老钥匙极为邪异，在齐瑯琊跟踪狐计的第三天，狐计得到了它，似乎早有预料一般，它散发出来的邪异力量狐计无法控制，将其临时封存在提前准备好的金黄匣子之中，不成想被齐瑯琊窃取。
   盗墓贼利用盗法将宝贝盗出，窃贼利用窃法将宝贝窃走。
   狐计的努力，为他人做了嫁衣，定然不肯善罢甘休，虽天地广阔，人海茫茫，但金字塔主人与法老钥匙必然存在联系，等狐计将金字塔主人经由自己的傀儡神通炼制为傀儡，就能操纵傀儡寻着气息找来。
   不过金字塔主人应当是一个极为厉害的人物，齐瑯琊亲眼看见狐计炼制时的艰难，初始阶段就屡次失败，最接近成功的一次，还被齐瑯琊看准时机窃走法老钥匙，而不得不中断。
   但就算他找来，齐瑯琊也不惧他。
   齐瑯琊单手僵尸化，血色潮汐涌现，像是人血汪洋流动在小小的一手之中，紧扣住金黄匣子，要将其撕裂，释放不详。
   但金黄匣子坚固无比，连冷硬的铁锁链网都能轻易撕裂的力量竟对金黄匣子丝毫不起作用，连细致的划痕也无法留下。
   他抚摸着浮雕表面，细细的思索着，浮雕应当是某种阵法或结界的载体，或特殊的机关，需要破解浮雕才能破解匣子，那就需要一名精通机关的偃师。
   “咚！”“咚！”
   门外传来了敲门声。
   “客人，你点的饭食”一个女子的声音传了进来。
   齐瑯琊将金黄匣子绑在一张青灰色的布里，拉开门栓，打开推拉门，一名美貌的女侍者两手端着一张木质托盘走了进来，将三碟菜肴和五碗米饭放下，对着齐瑯琊鞠了一躬，恭敬的拉上门退了出去。
   齐瑯琊很快将饭菜吃完，吹动从房中顶壁垂落至圆桌中央的口哨，女侍者听着呼唤前来，收起碗碟，将饭桌打扫干净，又鞠了一躬，退了出去。
   他坐在圆桌旁，饮茶数杯，尤觉饥渴难耐。
   齐瑯琊进入森罗惧森五天后遇见的狐计，又跟踪了狐计九天，已经有十四天没有饮用人血了。
   作为一具有自主意志、能够自由转换原来的人类身体和僵尸身体的僵尸，他不需要像普通僵尸一样单凭本能的觅食，但如果长期不饮用人血，他体内的力量就会下降，下降到最低点时，各方面的能力都只有原来的三分之一左右（他做过三次实验，误差值几乎为零）。
   他将绑着金黄匣子的布绑在腰间，随身携带，打开门，用钥匙将推拉门锁上，出去寻找人血。
```

# 第二回   雨中少年

```
   下午时分，乌云笼罩，天色暗淡。
   “轰隆！”“轰隆！”……
   紫色的雷霆在云间滚动，树枝状的雷电时不时的闪现，龟裂了天空，发出极致耀眼的光芒。
   遽然间，白光剧闪，齐瑯琊的视野刹那纯白，时间极短暂，大约不到半个呼吸的时间后：
   “哧！”
   震耳欲聋的闪电劈落声传来，四周空气都为之一滞。
   齐瑯琊的身体燥动起来，僵尸的唯一弱点是雷电，雷电能对僵尸造成极大损伤，普通僵尸没有自主意志，本能的极度畏惧雷电，每逢打雷的雨天，都会尽力躲避，山洞中，河水里，大树下，泥坑底，或者慌乱逃窜。
   齐瑯琊镇压燥动，走上街道。
   几点雨滴打落，很快急促，倾盆大雨泼洒下来，街道行人纷纷躲避，只有齐瑯琊雨中前行。
   雨水让他湿透了全身，从他脸上滑落，眼睛，嘴巴，鼻子，耳朵，全都有水流滑动。
   一头湿透的黑发，像是湿透的乌鸦羽毛般，披洒在颈后。
   他在雨中行走，如同历经一场神圣的仪式，心灵晋入空灵之境，寻觅目标的目光幽幽的转动，一瞬之间，进入深度心流状态。
   大雨磅礴，街上行人都在避雨，湿漉漉的堆积在一起，像是等待着发霉的谷物，只有执着雨具的人快步跑回家，或者就近加入一支发霉的队伍。
   没有落单的厄运儿。
   齐瑯琊如此总结。
   “哗啦！”“哗啦！”……
   雨水很快在街道上堆积，抬脚抬低了些就要撞水而行，齐瑯琊不知疲倦、不知劳累、不知雨滴与轰雷，走着。
   有人与他一样，那是一个少年，八九岁模样，他踡缩在街边，双手紧紧横抱着自己的双膝，下巴放在双膝之间，双目失神的望着雨中的景象，水已经淹过了他的脚踝，身旁一个像是老旧床板的发霉木牌上端端正正的刻着：卖身葬父。
   齐瑯琊看见他，像是看到了曾经的自己，世界仿佛空旷了，只剩他与那个少年，但那个少年消失了，一个更小的孩童出现了，同样的动作，踡缩着，狠狠的踡缩着，像是自己把自己抛弃进泥土里，狠狠的埋葬，永不醒来，就此死去！
   “你在干什么！”他大吼，在雨中，一瞬之间，以一种不可遏制的力量，但他最终清醒过来，一瞬之间的失神。
   那个少年看向了他，脸上的皮肉抖动着，寒冷么，惊颤么，还是，两者都有？
   少年不语，转头继续看着空旷的街道。
   齐瑯琊看着他，时间仿若静止了一般。
   “我买下你”他轻声说道，但就是这么轻的一道声音，却轻易的穿透了雨声，清晰的传入少年的耳中，令他猛然转头，直愣愣的盯着齐瑯琊。
   少年姓云，单字名，八岁，家境贫寒，母亲在他四岁那年死于疾病，父亲五天前因劳累过度而猝死，独留下他一人，因为买不起棺材，又不愿草席裹尸草草入土，只好卖身葬父。
   他说，这是父亲生前的执念，活着艰难，死时也要有个居所，权当劳苦一辈子的归宿。
   齐瑯琊带着他去死者商道屋买了一副上好的棺材，刻好碑文，将棺材石碑一并收入纳空戒。
   走到贫民窟，进入一间杂乱的拥挤破屋，齐瑯琊就看见一具腐败的尸体横陈在一张草席上，散发着刺鼻的尸臭味。
   把尸体装进棺材，出了城，走到云名母亲埋葬之土，一处无主之地，就旁掘开泥土，把棺材放进去，堆好土丘，插上石碑。
   云名浑身颤抖着，看着旧坟新坟，劳苦父母最终的死地，不禁坠下泪来，嚎啕大哭。
   哭了许久，才依依不舍的跟着齐瑯琊离开。
   在走回酒楼的路上，齐瑯琊抹去了云名的姓氏，赋予云名一个新的名字：剑流。
   “从今往后，我为主人，你为侍从”
   “是，主人”剑流改口很快，从“你”变为“主人”，只经历了不到一个时辰的时间，但对于剑流而言，这是一个命运的转折点，而跟随着一位修士，似乎也昭示着是向好的方向转折。
   哪怕在他换下破布衣裳，吃饱喝足之后，齐瑯琊抽了他一碗血并当面喝下。
   …………
   第二天一早，恰好在卯时四刻，齐瑯琊从床上醒了过来，他对时间的流动很敏锐，除非是全身心的投入战斗或修炼，否则他对时间的感知很少错乱，昨天他修炼到丑时睡下，睡到这个时间节点正好是两个半时辰。
   几乎是在他醒来的同一时间，门外响起了敲门声。
   “主人，醒来了么？”
   “嗯”齐瑯琊应了一声，他叫剑流在卯时四刻来敲他的门，他果然没有让他失望。
   齐瑯琊推开推拉门，问道：“你识字吗？”
   身高只到齐瑯琊颈下的剑流点了点头：“识字”
   他继续问道：“3的3次方？”
   “27”
   “自学的？”
   “自学的”
   “很棒，自学的能力很重要”齐瑯琊笑道，接着话锋一转，又问道：“俯卧撑能做几个？”
   剑流一愣：“俯卧撑？”
   齐瑯琊一怔：“你不知道俯卧撑是什么？”
   “不，我知道，但我已经忘了我能做多少个了”
   齐瑯琊说道：“现在做做看”
   “是，主人”
   剑流四肢张开撑在地上，开始做俯卧撑。
   一个、两个、三个……五十个。
   “主人，五十个”
   “嗯”
   接着，齐瑯琊走出门去点菜，然后回到房中，坐在圆桌上对着剑流说道：
   “凡人想成为修士，就得用各种锻炼方法去锻炼身体，达到身体的极致，在这种又苦又累的熬炼中，将身体训练到完美的凡人状态，或某一部分超凡脱俗，灵魂之内的玉星云就会开启，玉星云的开启将能给你带来极大蜕变，从百年凡人变成千年修士，寿元翻上十倍，身体的力量、速度、柔韧性、爆发性、抗击打能力等等都将得到极大的增强。”
   “等我跟你去到鹰皇都后，我会给你制定一个训练计划。”
   剑流感激道：“谢主人！”
   然后又问道：“我跟主人要去鹰皇都吗？”
   齐瑯琊说道：“吃饱喝足后就走！”
   他想着，如果不是自己在森罗惧森遇见那个傀儡神通者，看见他以不知名的秘法令一座偌大的金字塔在巨树森森的地域中凭空现形，多拖了这几天时间，以他的性子，早就拿着收集好的炼制材料，在鹰皇都的某间药剂室里炼制毒药了！
   不过，这几天时间却拖得极有价值，堪称血赚！法老钥匙具体有什么作用他不知道，但能够判定出来的是，这绝对绝对是一个好宝贝！一座神秘金字塔的至宝，散发着极致不详，需要特别封存的事物！
   他的眼中闪过贪婪的意味，他需要去鹰皇都，找到一名偃术高手，破解金黄匣子！
```

# 第三回  鹰皇都

```
   吃完后，剑流看向齐瑯琊，怯怯的伸出右手：“主人，要喝吗？”
   齐瑯琊一怔，旋即想起昨天将剑流的父亲埋葬之后，又去买了些衣物，然后到医馆买了一副抽血器具，回到酒楼时已经是夜晚，昨天剑流狼吞虎咽了一顿晚餐之后，就被自己抽了一碗血来喝，所以他误会自己每吃一顿都要饮用来自自己侍从身上的新鲜人血了。
   他说道：“我三天喝一次就可以了”
   “是，主人。”
   “走吧”齐瑯琊率先起身，剑流紧随其后跟上。
   他对剑流保有着警惕之心，僵尸需要饮血，这是众所周知的事情，在一些被僵尸占据的地域中存在着极少数的拥有自主意志的僵尸也是许多人所知晓的事，拥有自主意志的同时，还同时拥有人类和僵尸两种身体形态的更是极少数中的极少数，齐瑯琊至今未曾见过除自己外的另一人/僵，只在史书和杂志上看到过。
   而这种极致稀少的存在，这种小城的居民可不一定知道，更别提在贫民窟中长大的剑流了，但不得不警惕，在他没有确认剑流的忠诚心脏的斤两之前，必须得留意他的举动，倘若发现他有报官的心思，可就别怪齐瑯琊心狠手辣，一次性抽干鱼塘，痛饮人血了！
   …………
   在奇衡世界的正东之地，有一大洲称为武部隐洲，其地域极为辽阔，大小国度高达数万，有火山连绵如尘世巨蟒隔绝中部地带，中部往北称北方，中部往南称南方，南方偏东，最北之处，有一大国，名为祖墨国。
   鹰皇都，就是祖墨国的首都。
   齐瑯琊现在所处的小城离鹰皇都有着数十万里的距离，如此遥远的距离，自然不可能依靠双脚赶路。
   齐瑯琊带着自己的侍从，乘上由一头大鳄鱼拉动的大车厢，抵达一座大城池，然后在大城池乘上巨大的飞空舰船去到另一座大城池，如此转乘了四次飞空舰船，共花费六天时间，才到达目的地——鹰皇都。
   鹰皇都作为祖墨国的都城，其繁荣与奢华自不必多说，占据的地域范围更是头等，朝中高官贵族至少有七成的宗家族地都坐落在此，像是横向扩散的雨水一样，积攒在万千生灵头顶上的乌云把自己的影响力从鹰皇都开始扩散至全国。

# 第四回  毒药

```
   七月的早晨，来到鹰皇都的第二天，齐瑯琊从自己戴在左手食指，暗红色，一匹狼将自己的身体卷动，尾巴抵在下巴，狼头上镶嵌着一枚银色琉璃的纳空戒中取出三颗红丸，往上一抛，张开嘴，正好落入嘴中，嚼都不嚼就直接咽了下去。
   此为齐瑯琊亲手炼制的毒药——九刑淋血丸。
   毒性相当猛烈，正常吸入即可中毒，更可溶于水，溶于水后无色无味，想要发挥其最强毒性的方式是直接服用。
   中毒者将全身剧痛，像是有人用凿子凿击四肢百骸，浑身战栗。
   皮肤呈红色到黑色之间的渐变色，并且渐变不止，鼻大出血。
   尿意强烈，尿出血，无法止住，尿流时下体器官将产生强烈刺痛感。
   在一刻钟内中毒者就会死于失血过多，死亡过程非常痛苦，全程哀嚎不止。
   齐瑯琊服毒不是要自杀，因为还没到死的时候。
   他身为僵尸，对毒药的抗性非常强，对同境修士来说绝对致命的毒药，他能完全免疫绝大部分，剩下的小部分就算不能完全免疫，发作时对他的损伤也较为轻微。
   但齐瑯琊并不满足于此，他身为一名毒师，自己能够炼制毒药，于是每天都会服用毒药借此来增强自己对毒药的免疫力，有时候玩大了只需要切换僵尸身体，凭着超高的恢复力和抗毒性，就能保着自己小命不失。
   在与剑流吃过早餐后，齐瑯琊就任由剑流去锻炼，自己离开酒楼，前往一个巨大的蜂巢状建筑，其名为“药剂蜂巢”，是一种经典的用于对外租用的药剂室集群造型，其内有着数千个药剂室，是不太富有或游走天下的炼丹师、毒师、医师的必去场所。
   齐瑯琊走入大厅，在前台交完钱后取过握权信物——一块玉牌，它是药剂室的钥匙，然后往其上标注的编号为“359”的药剂室走去。
   来至门前，药剂室的铜门上有一只上下闭起来的眼睛，拳头大小，感知到有人到来，即张开眼睛，向前直射出一道红色光束，齐瑯琊举起玉牌挡住那红色光束，不到三个刹那的时间那红色光束收回，黑色的眼珠子变为蓝色，药剂室的铜门自动打开，齐瑯琊走了进去，铜门自动关上，蓝色的眼珠子变为红色，睁得大大的，如同在警戒一般。
   其内是六边形的空间，摆满了各种金属器皿和炼制药剂时的必备器具，上方的六边形壁顶中央空出一块方盒大小的凹槽，有一块白色菱形水晶镶嵌在其中洒下耀眼的光芒。
   周围有六面黑色墙壁，每面墙壁上都有一个人头大小的涡轮，用于在毒气泄露时吸走毒气。
   齐瑯琊此番是要炼制一种名为“颠气”的毒气，这种毒气有强大的致幻效果，中毒者瞳孔扩散，面色潮红，四肢冰冷，身体极度困倦和疲惫，变得谵妄，出现意识障碍、感知异常、日夜颠倒等情况。
   中毒者还会恶心呕吐，初期步态蹒跚、脚步虚浮，不到一刻钟的时间就会流着口水四肢爬行，时不时的从喉咙里吐出呕吐物，持续超过五个时辰，然后在八个时辰以内陷入昏迷，心脏在昏迷中停止跳动，真正的死亡降临在中毒者的身上。
   齐瑯琊将颠气的炼制材料从纳空戒中取出，放置在白银台上，走到一尊青铜炼丹炉的面前，掀开炉盖，将白银台上买来的芦根豆、药罂、艾害子、斑虫草、寒枯参等药草称了称，按脑中记下的比例放入研钵中捣碎，碾成粉末，倒入炉中，再倒入一瓶他从森罗惧森的蓖乌树中取到的剧毒汁液，搅成黑色稠糊，盖上炉盖。
   齐瑯琊全神贯注，双掌伸出，释放火焰，精心控制和变化着火候，开始炼毒。
   约莫炼了半个时辰，掀开炉盖，炼丹炉中是一颗紫黑色的毒球，食指和拇指圈起来的圆圈大小，齐瑯琊用镊子夹出毒球，丢入冰鉴中冻上半个时辰，再夹进炼丹炉，把自己以前就收集到的一种有毒浆果——霍茄，往研钵中丢进六颗，捣碎碾成黏稠的果肉，倾进炼丹炉，再将一瓶蓖乌树的剧毒汁液浇在毒球上，往炉盖内壁中央挂一个能自动收集毒气的毒囊，盖上炉盖，关闭除进火口以外的其它孔洞，继续炼制。
   又炼了两刻钟，再开炉盖时，炉中的毒球果肉汁液都被炼成了颠气，毒囊鼓鼓囊囊的，颠气都被收集进了里面。
   齐瑯琊眉毛舒展，发出轻笑，一次成功，令他非常高兴，眼睛中有着难以掩饰的得意色彩。
   他怀着激动的心情，把毒囊收进纳空戒中，将炼制材料中最少的一种全部倒进研钵，其他的按比例也一并用上，打算一次性把能炼制的颠气都炼制出来。
   夜晚，齐瑯琊从药剂室离开，除了颠气外，他还炼了另外三种毒，其中就包括他日常服用的九刑淋血丸，装满了四个玉瓶。
   回到酒楼，推开剑流的房门，看见他一头蓝色短发下是疲惫的面容，两只手上，除两边大拇指外的拳峰和近指关节上都有伤，但眉宇间却坚毅了几分。
   齐瑯琊问道：“三轮锻炼都做完了么？”
   剑流认真的点了点头，报告道：“是，都做完了！”
   齐瑯琊说道：“万事开头难，一个锻炼计划的开头有两天，第一天是看你有没有毅力坚持得下去，第二天你会浑身酸痛，难以行走，走路时都会有一定的疼痛感，但只要坚持下去，进入锻炼状态，疼痛感会暂时消失，只要撑过第二天，剩下的就简单了。”
   “是！”
   齐瑯琊回到房间，进入浴室，解下腰间的金黄匣子，脱下衣服，卸下飞刀，解开手脚上的绷带，脱掉体表上的贴身内甲，拉动墙上木杆，竹管上有水流入浴桶中，齐瑯琊走入浴桶把身体上因为炼毒时的高温火焰炙烤而流下的汗液洗净，然后洗了个头，把衣服和绷带洗完，穿上内甲，换上昨天晾干了的绷带和黑色劲装，装载好飞刀和金黄匣子，走出浴室。
   端坐在床上修炼造根法，提炼自身，精细明确的渗透至肉身的每一缕血肉，如同把自己的身体当作一件兵器般去铸造炼化。
   忽然，他睁开双眼，感受到了一丝不详的味道。
